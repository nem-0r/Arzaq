# Quick Start Guide - Community Posts Feature

## Как использовать функцию создания постов

### 1. Запуск приложения

```bash
cd arzaq-react
npm start
```

Приложение откроется на http://localhost:3000

### 2. Регистрация

1. Перейдите на страницу регистрации: `/register`
2. Заполните форму:
   - Полное имя
   - Email
   - Пароль
3. Нажмите "Register"

### 3. Вход в систему

1. Перейдите на страницу логина: `/login`
2. Введите email и пароль
3. Нажмите "Login"

### 4. Создание поста

1. Перейдите на страницу Community: `/community`
2. Нажмите зеленую кнопку **"Create Post"**
3. В открывшемся окне:
   - **Выберите ресторан** из списка (можно искать по названию)
   - **Напишите текст** о вашем опыте (минимум 10 символов)
4. Нажмите **"Post"**
5. Ваш пост появится в ленте!

### 5. Если вы не авторизованы

При нажатии на "Create Post" появится окно с предложением:
- **Login** - войти в существующий аккаунт
- **Create Account** - создать новый аккаунт

## Структура файлов

```
src/
├── services/
│   └── api.js                 # API сервис (готов к подключению бэкенда)
├── components/
│   ├── common/
│   │   └── LoginPrompt/       # Модалка для неавторизованных
│   └── features/
│       └── Community/
│           ├── CreatePostButton/    # Кнопка создания
│           └── CreatePostModal/     # Форма создания поста
└── pages/
    └── CommunityPage/         # Страница Community
```

## Особенности

✅ **Проверка авторизации** - только авторизованные пользователи могут создавать посты
✅ **Валидация** - пост должен быть минимум 10 символов
✅ **Выбор ресторана** - обязательно указать заведение
✅ **Поиск ресторанов** - легко найти нужное место
✅ **Реальное время** - посты сразу появляются в ленте
✅ **Responsive** - работает на всех устройствах

## Готово к production!

Код написан так, чтобы легко подключить настоящий бэкенд:
- API сервисный слой уже готов
- Все функции задокументированы
- Обработка ошибок реализована
- Валидация на месте

Просто раскомментируйте секции с `// BACKEND:` в `src/services/api.js` и подключите ваш API!
